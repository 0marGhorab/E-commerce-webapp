# E-commerce-webapp
A responsive e-commerce web application built with React, showcasing products, categories, and cart functionalities. It consumes a RESTful API for product data, user authentication, and order management. Ideal for demonstrating front-end integration with a real-world API.  API used: E-Commerce API Documentation

🛍️ E-Commerce Frontend App
A responsive and user-friendly e-commerce web application built with React. This project showcases product listings, categories, shopping cart features, and user authentication by integrating with a RESTful API.
